Git/Github
Git != GitHub
Git - это система контроля версий
GitHub - это облачное хранилище исходных кодов

Repository - это хранилище исходных кодов.
fetch - скачивает и НЕ распаковывает изменения с удаленного репозитория в локальный
pull  - качивает и распаковывает изменения с удаленного репозитория в локальный
push  - отправляет изменения с локального репозитория в удаленный

Commit - это контрольная точка, в которую сохраняется состояние всех нужных файлов репозитория.
Commit хранит снимок репозитория.

Branch
merge  - это слияние (когда commit из одной ветки вливается в другую ветку)
rebase (пересадка) выполняется когда ветку хотят "пересадить" на другой commit.
