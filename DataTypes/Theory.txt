Переменная (Variable) - это именованная область памяти,
содержимое которой может изменяться в процессе выполнения программы.

				type name;	//синтаксис объявления переменной
							//declaration of a variable
type - тип переменной
	1. Сколько памяти переменная будет занимать;
	2. Какие значения она сможет принимать;
	3. Какие операции над ней можно выполнять;

I)	Логические типы: 
	bool (Boolean).
	Переменная типа bool занимает 1 Байт памяти, и принимает одно их двух значений:
		true	1
		false	0
II)	Символьные типы: char (Character - Символ) 1 Byte ASCII
	ASCII - American Standard Code for Information Interchange
	256
III)Числовые типы делятся на целочисленные и вещественные
	unsigned - могут хранить лишь положительные целые числа
	  signed - могут хранить как положительные, так и отрицательные целые числа
	  ------------------------------
	  short
	  long
	  int
	  long long
